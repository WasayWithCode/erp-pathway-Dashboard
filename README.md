# ERP Pathway Dashboard

A modern ERP (Enterprise Resource Planning) Dashboard built with React, featuring analytics, user management, reports, and a responsive admin interface.

## 🚀 Features

- Modern Dashboard UI
- Responsive Design
- Sidebar Navigation
- User Management
- Analytics & Reports
- Authentication System
- API Integration
- Role-Based Access Control
- Real-Time Data Visualization
- Clean and Scalable Architecture

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Tailwind CSS
- Axios
- React Router
- Node.js (Backend)
- Express.js
- MongoDB

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/WasayWithCode/erp-pathway-Dashboard.git
```

### Navigate to Project

```bash
cd erp-pathway-Dashboard
```

### Install Dependencies

Client:

```bash
cd client
npm install
```

Server:

```bash
cd server
npm install
```

## ▶️ Run Application

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
npm start
```

## 🌐 Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
